## 1. ใช้งาน (สำหรับเจ้าหน้าที่)

ใช้ได้ทั้ง **มือถือ / แท็บเล็ต / คอมพิวเตอร์** (หน้าเว็บปรับตามขนาดจอเอง)

1. เปิดเว็บ → ใส่ **ปีการศึกษา** (จำค่าไว้ให้ครั้งต่อไป)
2. เปิด/ปิด **สวิตช์** ตามต้องการ
   - `แยกโฟลเดอร์ตามฝ่าย/หมวด` — สร้างโฟลเดอร์ของแต่ละหมวดแยกในปีนั้น
   - `แปลง PDF อัตโนมัติ` — บันทึกทั้ง Excel และ PDF เข้า Drive
   - `รายงานตามแบบราชการ` — ใช้ตาราง ด้าน/รายการ/x̄/S.D. ตามแบบตัวอย่าง (เฉพาะไฟล์ตระกูล TQF)
3. **แตะหรือลากไฟล์** .xlsx/.xls/.csv (หลายไฟล์พร้อมกันได้) → **แถบความคืบหน้า** แสดงทุกขั้น
4. ดูผล (จำนวนผู้ประเมิน · x̄ · S.D. · แปลผล · แยกชั้นปี)
5. เลือกทำอย่างใดอย่างหนึ่งหรือทั้งหมด
   - `⬇ Excel` — ดาวน์โหลดไฟล์ Excel ลงเครื่อง
   - `⬇ PDF` — เปิดหน้าพิมพ์ → เลือก “บันทึกเป็น PDF”
   - `☁ บันทึกเข้า Google Drive` — เก็บทั้ง Excel + PDF เข้าโฟลเดอร์อัตโนมัติ พร้อมลิงก์เปิดโฟลเดอร์

### โครงโฟลเดอร์ที่ระบบสร้างใน Drive
```
งานประเมินจากเว็บ_ปีการศึกษา_2568   (โฟลเดอร์แม่)
└── ปีการศึกษา 2568
    ├── นภาภิบาล การฝึกร่วม 4 ชั้นปี     ← แยกตามหมวด/ฝ่าย
    │   ├── ผลประเมิน_....xlsx
    │   └── ผลประเมิน_....pdf
    └── ศิลปวัฒนธรรม ชมรมดนตรีไทย
        └── ...
```
- ไฟล์ชื่อซ้ำ **ไม่เขียนทับ** ระบบเติมเวลาต่อท้ายให้เอง
- ระบบ **ไม่ลบ** ไฟล์/โฟลเดอร์เดิมใดๆ

### PDF ภาษาไทยสวย
ติดตั้งฟอนต์ **TH Sarabun PSK** ในเครื่องก่อน แล้วตอนกด `⬇ PDF` เลือก “บันทึกเป็น PDF”
(ถ้าไม่มีฟอนต์นี้ ระบบจะใช้ Sarabun/Tahoma แทน อ่านออกแต่หน้าตาต่างเล็กน้อย)

## 2. หลักการคำนวณ

| หัวข้อ | กติกา |
|---|---|
| มาตรวัด | Likert 5 ระดับ |
| ค่าเฉลี่ย | x̄ ต่อข้อ / ต่อชั้นปี / รวม |
| ส่วนเบี่ยงเบน | **STDEVP** (แบบประชากร) ให้ตรงกับไฟล์ตัวอย่างเดิม |
| นับข้อ | **เฉพาะข้อย่อย** — ข้อแม่ (เช่น 6.1 เมื่อมี 6.1.1) ไม่นับซ้ำ |
| แปลผล | 4.50–5.00 มากที่สุด · 3.50–4.49 มาก · 2.50–3.49 ปานกลาง · 1.50–2.49 น้อย · 1.00–1.49 น้อยที่สุด |

> รายงาน **แบบราชการ** แสดงครบทุกข้อ (รวมข้อแม่) ตามไฟล์ตัวอย่าง — แต่ **ค่าเฉลี่ยรวม** คิดจากข้อย่อยเท่านั้น

## 3. ตระกูล FORMAT (สำคัญที่สุด)

ไฟล์ประเมินแต่ละหมวด/ปี **โครงสร้างไม่เหมือนกัน** ระบบตรวจอัตโนมัติแล้วเลือกวิธีอ่านให้ถูก

| ตระกูล | หมวด | โครงสร้าง | สถานะ |
|---|---|---|---|
| **A** | นภาภิบาล, เจตคติ | ชั้นปีแยก sheet · 44 ข้อ · 8 ด้าน TQF | ✅ ใช้ได้ |
| **B** | ศิลปวัฒนธรรม, ชมรม | sheet เดียว · **ชั้นปีอยู่ในคอลัมน์** · 20 ข้อ · มี X/SD มาแล้ว | ✅ ใช้ได้ |
| **E** | อาจารย์ที่ปรึกษา | ชั้นปีแยก sheet · 11 ข้อ · มีข้อเสนอแนะ | ✅ ใช้ได้ |
| **C** | ผู้สอน/ครูฝึกรายบุคคล | ประเมินผู้สอนแยกคน | ⬜ ยังไม่รองรับ |
| **D** | สมรรถนะตาม LO/มคอ. | PLO/CLO | ⬜ ยังไม่รองรับ |

### กติกาตรวจอัตโนมัติ (เรียงลำดับ)
1. มีคอลัมน์ชื่อมี “ชั้นปี” → **ตระกูล B**
2. มี sheet ชื่อ ปี4/ปี3/ปี2/ปี1 → ชั้นปีแยก sheet (A/E)
3. 44 ข้อ + มีรหัสย่อย (x.y.z) → **ตระกูล A** (ใช้ mapping 8 ด้าน TQF)
4. ~11 ข้อ + มีคอลัมน์ข้อเสนอแนะ → **ตระกูล E**
- ถ้าไม่ชัด ระบบแสดงป้ายบอกว่าตรวจได้เป็นตระกูลอะไร — ตรวจก่อนใช้ผลเสมอ

## 4. โครงสร้างโค้ด (สำหรับผู้ดูแล)

```
index.html          ← ไฟล์เดียวจบ (SheetJS + TQF mapping + engine ฝังในตัว ไม่พึ่ง CDN)
SKILL.md            ← สเปกระบบ + กฎเหล็ก (อ่านก่อนแก้ทุกครั้ง)
WIKI.md             ← ไฟล์นี้
dev/
  engine.js         ← สมองคำนวณ: parseWorkbook / compute / buildExportAOA
  shell.html        ← หน้าเว็บ (UI)
  build.py          ← ประกอบ index.html = shell + SheetJS + tqf.js + engine
  tqf.js            ← mapping 8 ด้าน + 44 ข้อ + เกณฑ์ (ดึงจากไฟล์ตัวอย่างจริง)
gas/
  uploader.gs       ← Apps Script เก็บไฟล์เข้า Drive (แยกโฟลเดอร์ + แปลง PDF)
```

### แก้แล้ว build ใหม่
```bash
cd dev && python3 build.py     # → index.html
```
แล้วอัป `index.html` ขึ้น repo (GitHub Pages อัปเดตเอง)
> ⚠️ `index.html` ~670KB เพราะฝัง SheetJS — **อัปจากคอมพิวเตอร์** จะชัวร์กว่ามือถือ

### เพิ่มตระกูลใหม่ (เช่น C ผู้สอน)
1. **เปิดไฟล์จริงดูก่อน** — header อะไร, ชั้นปีอยู่ไหน, กี่ข้อ, มี X/SD มาแล้วไหม
2. เพิ่ม branch ใน `parseWorkbook()` (engine.js)
3. เพิ่มกติกาตรวจใน `detectFamily()` (shell.html)
4. ถ้ามีแบบรายงานเฉพาะ → เพิ่ม mapping แบบ tqf.js
5. **ทดสอบ regression** — ตระกูลเดิมต้องได้ผลเท่าเดิม (นภาภิบาล = 939 คน x̄ 4.51)

## 5. แก้ปัญหาที่พบบ่อย

| อาการ | สาเหตุ | วิธีแก้ |
|---|---|---|
| ลากไฟล์แล้วเงียบ ไม่ขึ้นผล | ใช้ index.html เวอร์ชันเก่า (โหลด SheetJS จาก CDN) | อัป index.html ตัวใหม่ + กด Ctrl+Shift+R ล้าง cache |
| กดบันทึก Drive แล้ว error “ตอบกลับไม่ใช่ JSON” | Apps Script ไม่ได้ deploy แบบ Anyone | Deploy > Manage deployments > แก้เป็น Anyone |
| PDF ภาษาไทยเป็นกล่อง | เครื่องไม่มีฟอนต์ไทย | ติดตั้ง TH Sarabun PSK |
| ผลออกมาข้อไม่ครบ/เกิน | ระบบตรวจตระกูลผิด | ดูป้ายตระกูลบนการ์ดผล — ถ้าผิด แจ้งผู้ดูแลเพิ่ม parser |

## 6. Apps Script (uploader)

ตั้งค่า: `PARENT_FOLDER_ID = 1b9213ym4rhJjs9IoZUV9AKJrBpE2kCoR`

| API | ทำอะไร |
|---|---|
| `doGet()` | ทดสอบเชื่อมต่อ → คืน `{ok:true, folder:"..."}` |
| `doPost()` | รับ `{filename, year, category, makePdf, reportHtml, base64}` → สร้างโฟลเดอร์ + เก็บ Excel + PDF → คืนลิงก์ |

**ความปลอดภัย:** โค้ดมีแต่ `createFile` / `createFolder` — **ไม่มีคำสั่งลบหรือเขียนทับ** ไฟล์เดิมทุกไฟล์ปลอดภัย
